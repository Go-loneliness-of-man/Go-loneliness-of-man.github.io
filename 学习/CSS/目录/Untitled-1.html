CSS 过渡

一、CSS 过渡

CSS 过渡是个比较好玩、好看的属性，它是通过 CSS 伪类选择器来触发的。
属性描述属性值
transition-property选定该元素要进行过渡的属性none、all、单独写上各个属性
transition-duration指定过渡所需要的时间时间单位， s
transition-timing-function指定过渡的方式ease、linear、ease-in、ease-out、ease-in-out、cubic-bezier()
transition-delay指定开始过渡的延迟时间时间单位， s
transition简写形式顺序是属性、过渡时间、过渡方式、延迟时间
transition-property、transition-duration 用法很简单，这俩的作用依次是选定过渡属性、指定对应属性的过渡时间，拿一个 div 元素来展示（hover 触发）：

第一个 CSS 选择器里前几行都是设置静态样式的，CSS 过渡属性位于最后两行，倒数第二行选定了要进行过渡的属性，倒数第一行指定了过渡所需要的 时间。第二个 CSS 选择器指定了元素过渡后的末状态，即指定了要进行过渡变化的 CSS 属性的末状态。

代码如下： #gd1{ display:block; float:right; color:rgb(10,10,10); width:80px; margin:10px 600px 10px 0px; background:white; padding:20px 30px; border-radius:30px; transition-property:all; transition-duration:0.8s; } #gd1:hover{ background:black; border-radius:90px; padding:20px 100px; color:rgb(220,220,220); margin:10px 550px 10px 0px; }
transition-timing-function、transition-delay 依次决定的是变换方式、变换延迟。

二、CSS 动画

CSS 动画比 CSS 过渡更灵活、更强大，它不必通过触发演示，默认下是自动播放的，并且它用关键帧来实现动画，说白了就相当于为 CSS 过渡加上多个状态（过 渡只有始、末两个状态），下面是 CSS 动画的属性一览：
属性属性值描述
animation-name@keyframes 规则的名称选择该元素执行哪个关键帧动画
animation-duration以 s 为单位与过渡一样
animation-timing-function与过渡一样同上
annimation-delay以 s 为单位同上
animation-iteration-count数字、infinite指定动画的循环次数
annimation-directionnormal、alternate确定动画是否交替播放，即正向一次，反向一次
animation-play-statepaused暂停动画
animation-fill-modenone、forwords、backforwords、both确定动画播放后元素的状态
animation简写形式顺序是 name、duration、function、循环次数、是否交替、delay、mode
@keyframes nameCSS 属性定制一段动画，确定其在不同进度时的关键帧
@keyframes、name、duration @keyframes 定制关键帧，name 引用它，duration 确定动画演示的时间，这样就组成了动画演示的基本要素。下面做个例子：

解释：

@keyframes 确定的动画名称可以与元素的 id 重名，#dh1 规则的最后两行确定了该元素要引用哪个动画、动画持续的 时间，@keyframes 规则定制了一段动画，这段动画分 5 个关键帧（帧数可任意指定），每个关键帧都对要进行改变的 CSS 属性进行了设置，实际上，每 两个关键帧之间就相当于进行一次 CSS 过渡，
