title属性和id属性、target属性、几个新标记

一、title属性和id属性

一个好的网站必须要做到人性化，这是它与普通网站的差距，对网站进行“抛光”的方法有很多种，现在先介绍一种改进链接的方法。

title属性

如果你要单击一个链接时，能够了解关于它的更多信息就好了。这对那些有视力障碍使用屏幕阅读器的人来说尤为重要，为了保证标签的简洁，链接的标签 通常只能提供有限的描述，这样就用到了title属性，a元素的title属性是用来描述链接的，是的，它与head里的title元素重名。
对于大多数浏览器来说，鼠标移到一个浏览器上时，会有一个工具提示，那就是title属性做的，不过对于视力障碍的人来说，浏览器可能会大声的读出链 接的title属性。其实可以为任何元素添加title属性，并且title属性对于有些元素还有其它用途。

id属性

对于篇幅比较长的页面，我们经常需要直接链接到页面的某个特定位置，这就需要通过id属性来实现。
id属性是一个重要的属性，它还有很多别的特性，标识元素只是其中之一，你可以直接使用a元素链接到被id属性标识的元素。
被标识的元素被称为目标元素，你需要为目标元素起一个标识符名，并在元素的开始标记中插入一个id属性，像这样<h2 id="WC">wangchao</h2 >,不论是相对链接还是URL，要链接到页面中的一个特定目标，只需要在链接的最后加上“#标识符名”就可以让a元素链接到它，像这样: <a href="index.html#WC">标签</a>。要注意的是在目标元素所在的页面其标识符名必须是唯一的，还有标识符名可以使用大小写英文字母 、数字、横线、下划线、冒号、点号来组合，但必须要以字母开头，最好都是大写或都是小写，不然链接就可能在某些浏览器上无法正常工作。
id属性可以用来为一个长页面创建目录，也可以为一个a元素增加id属性，这样你就可以链接到一个链接。
一个难以发现的错误是，当你写一个这样的地址时:“http://www.wangchao.com/wangchao#WC”,一定会使这个链接无法正常工作，因为结尾是目录， 所以浏览器会自动帮你在URL末尾加斜线，这样就取代了id引用。不过你可以自己在末尾加一个斜线来避免这种情况:“http://www.wangchao.com/wangc hao/#WC”。

二、target属性

target属性值会告诉浏览器页面的“目标窗口”，指示浏览器在“哪里”打开链接，如果使用“_blank”作为目标，浏览器就总是打开一个新窗口来显示页面，令打 开的链接出现在另一个窗口中。具体像这样使用:
<a target="_blank"
href="http://www.wangchao.com">wangchao</a>
现在大多数浏览器都用同一窗口下的不同标签页来取代打开新窗口，还在使用新窗口的只有少数像IE浏览器等。
用target指定一个特定的名字时，其实就是对显示链接页面的新窗口的命名，所有具有相同目标名的链接都会在同一个窗口中打开，“_blank”是一种特殊情况， 它告诉浏览器总是使用一个新窗口。

三、几个新标记

  1. <code>,用来显示计算机程序代码。
  2. <time>，告诉浏览器这个内容是一个日期或时间。
  3. <pre>，使浏览器原样输出标记的内容。
  4. <strong>，使用这个元素来标记你希望特别强调的文本。
