一、特定性的计算方法

CSS是层叠样式表，浏览器在显示页面的每个元素前会把所有对这个元素有定义的样式表都找出来，然后进行特定性的计算，最后决定对这个元素使用哪种样 式，特定性的计算方法可以按照它包含的id、类或伪类、元素名来进行分值计算:
id名100分，类或伪类10分，元素名1分。
由此可见，使用id名是最特定的，其次是类或伪类，然后是子孙选择器，最后是单个元素名。如果没有为元素设定样式，则会使用浏览器默认的样式。
还有一种情况是，如果在一个样式表下加上!important;这行，其下方的规则的特定性最高。像这样:

        h1{
            color:red;
            !important;
            color:green;
           }
