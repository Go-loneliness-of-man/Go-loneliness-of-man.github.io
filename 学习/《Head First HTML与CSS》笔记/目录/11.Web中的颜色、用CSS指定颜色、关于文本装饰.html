Web中的颜色、用CSS指定颜色、关于文本装饰

一、Web中的颜色

Web颜色类似于RGB颜色模式，也是由红、绿、蓝三种颜色混合而成的，每种颜色可以被指定从0到100%的数值，例如三种颜色都是百分之百就是白色，对应 的百分之零就是黑色，由于屏幕是发出光而不是反射光，所以将颜色混在一起就会得到更亮的颜色。
Web安全颜色:在浏览器的早期阶段，很少有能支持大量颜色的计算机屏幕，所以当时创建了Web安全颜色以确保大多数显示屏都可以显示。然而如今，情况已 经发生了巨大变化，大多数用户的计算机屏幕都支持上百万种颜色，所以现在基本可以不用再考虑Web安全颜色。
一般来说，让文本使用深色，背景使用浅色是比较好的搭配，当然反过来也可以。

二、用CSS指定颜色

用CSS指定颜色有四种方法:颜色名、红绿蓝百分数、红绿蓝数值、十六进制码。虽然有四种方法，但它们的原理是一样的，都是设定红绿蓝三种色彩的混合比例。
颜色名:即使用颜色的英文名，不区分大小写，这些颜色名只是预先定义了红绿蓝三种颜色分量是多少。
百分数:background-color:rgb(12%,43%,64%);依次定义的是红色、绿色、蓝色三种分量的百分比。
数值:和百分数基本一样，只是用0到255来指定一个颜色的比例，这样写:background-color:rgb(45,12,154);
十六进制码:以#开头，每两位表示一个颜色的分量，如#125y3f就是一个十六进制码，它只是将数值的0到255用一个两位十六进制数来表示,十六进制码可以缩 写，形如#ccaa33就可以缩写为#ca3。

三、关于文本装饰

对于同一个元素，同一个属性只能使用一次，也就是说如果你用同一属性对一个元素定义了两次，那结果只是选其中之一。不过你也没必要那么做，可以在一个属 性里设置多个参数，例如这样:h1{ text:overline underline; },就可以同时设置上划线和下划线，只需要在参数间用空格隔开即可。
一个新标记:<del>,删除线，它能为其中的文本都加上删除线。
