一、学习方法

你的大脑总是在渴求一些新奇的东西，我们每天都会遇到许多按部就班的事情，这些事情都很普通，对于这样一些例行的或者 说平常的事情，大脑不会予以太多注意力，它会自动过滤掉这些事情，大脑真正的工作是记住那些它自以为确实重要的事情。那 么大脑是怎么知道到底哪些东西是重要的呢？打个比方，假如突然有一只大老虎跳到你面前，此时此刻，你的大脑会作何反应？
神经元会“点火”，释放一些化学物质，接着情绪爆发.......
好了，这时候你的大脑就会知道......
这肯定很重要！可不能忘记了！
事实上，除非你很积极的让神经元活动起来，否则你的头脑里什么也不会发生。必须试着去解决问题、得出结论、产生新的知识。 情绪对于记忆也是一个很重要的因素，譬如是你关心的东西，就很容易记住，或者对你有所触动的东西，也容易留在你的脑海 里。还有自豪感，如果你解决了一个难题，学会了所有人都觉得很难的东西，或者发现你了解的一些知识竟是那些自以为无所不能 的傲慢家伙所不知道的，此时就会有一种自豪感油然而生。
如果你真的想学，而且想学的更快、更深，就应该注意你怎样才能专注起来。技巧在于要让你的大脑认为你学习的东西确实很重要， 对你的生活有很大影响，就像老虎突然出现在面前一样。对此有两种方法，重复和令你的大脑活动起来（熟能生巧）。
建议:

  1. 慢一些，尽量理解，理解的越多，需要记得就越少。
  2. 定时的做练习，做笔记总结。
  3. 保持充分的睡眠，其实长久的记忆是在睡梦中完成的。
  4. 多喝水。提供充足的液体，大脑才能有最佳表现，如果缺水，学习能力就会下降。
  5. 注意大脑是不是累了。如果你开始浮光掠影的看，刚看的东西就忘记了，这就说明你该休息一会了。
  6. 专注。一旦开始学习，就要保持专注，否则接下来必然会效率底下。

二、Web页面的显示

浏览器显示页面的过程其实很简单，一个页面的显示大体分三个步骤，一是浏览器向Web服务器发送资源请求；二是服务器收到请求后 查找资源然后发送给浏览器；三便是浏览器根据收到的资源（HTML文件、图片、音频等）来布置页面。
其中，HTML即超文本标记语言，根据HTML文件，浏览器可以得知页面的所有内容和结构。Web服务器就是一个储存资源的全天候 等待资源请求的计算机。

三、一些标记

  1. <html>,声明一个HTML文件的开始。
  2. <head>，页面首部的开始，页面的首部用于定义页面的一些基本属性。
  3. <body>，页面主体的开始，页面主体即浏览器显示的内容
  4. <h1>到<h6>，页面主体中的标题，分六个大小。
  5. <title>，浏览器页面的标题。
  6. <p>，一个段落的开始。
  7. <em>，斜体。
  8. <!--......-->,HTML中作为注释的标记。
HTML中绝大多数标记都是成对存在的，像这样<标记名></标记名>。
