三、程序阅读题

1. 下面这个程序的输出结果是_____。 void main() { int x, y, z = 2; x = z++; y = ++z; printf("%d %d %d",x,y,z); }
解析: 这道题要注意的就是自增运算符的“自增“与“取值”的顺序，x = z++; 是先取值再自增，即 z 先把值赋给 x 然后 z 再自增一次，y = ++z; 相反，z 先自增一次然后才把值赋给 y

2. 以下程序的输出结果是_____。 void main() { int a = 1, b = 2, c = 3; if( a < b ) if( a < c ) printf("%d\n",a); else printf("%d\n",b); }
解析: 这是一个选择嵌套，如果 a 既小于 b 又小于 c，就输出 a，否则输出 b。

3. 以下程序的输出结果是_______。
void main() { int a[10], i, b = 0; for( i = 0 ; i < 10 ; i++ ) a[i] = i; for( i = 1 ; i < 4 ; i++) b = b + a[i] + i; printf("%d\n",b); }
解析: 这个程序含两个循环，先看第一个，i 从 0 自增到 9，a[i] = i，所以循环结束后数组 a 每个数组元素的值等于它自身的编号（从 0 到 9）。接下 来看第二个循环，第二个 for 共循环三次，然后循环体是 b= b + a[i] + i; 相当于每次循环 b 自增 a[i] + i 这样大的值，而循环三次就是，从 0 开 始，自增三次。最后输出 b 的值。这题最好写在纸上，一步一步算。

4. 以下程序的输出结果是______。 void main() { char *p, a[] = { "0123456789" }; p = a; p++; printf("%s",p); }
解析: 这题考察了三个知识点，指针指向数组、字符串、地址运算，首先它声明定义了一个字符串 a，然后将 a 的首地址赋值给指针变量 p，最后 p++ 了一次， 那么 p 就指向了字符串的第二个元素 a[1]，也就是字符 1，所以利用占位符 %s 输出时从字符 1 开始输出。

5. 以下程序的输出结果是_____。 int fun(int x,int y) { int t; if( x < y ) t = x; else t = y; return t; } void main() { int a = 1, b = 2, c; c = fun( a , b ); printf("%d\n",c); }
解析: 先分析函数 fun(int x,int y) 的功能，它利用一个 if-else 语句实现了输出两个形参中较小的一个（其实我觉得它可以用三目运算符代替，这样代 码量更短），再看主函数，声明了a，b，c 三个变量，然后将 a，b 传入 fun() 中，将 fun() 的返回值赋给 c，输出 c 的值。

程序阅读题答案
  1. 2, 4, 4
  2. 1
  3. 12
  4. 123456789
  5. 1

四、编程题

1. 从键盘输入圆的半径 r，计算并输出圆的周长和面积。

答案 #include<stdio.h> #define PI 3.14 //定义一个宏表示圆周率 int main() { double r, c, s; //依次代表半径、周长、面积 scanf("%lf",&r); //获取半径 c = 2 * PI * r; s = PI * r * r; printf("周长:%lf\t面积:%lf\n",c,s); return 0; }

2. 编写程序，输入 3 个整数，输出其最大值。

答案 #include<stdio.h> #define MAX(x,y) ( (x) > (y) ? (x) : (y) ) //定义一个找出两数中较大者的有参宏 MAX int main() { int i, nums[3], max; //依次是循环变量 i、存储数列的数组 nums[]、最大值 max for( i = 0 ; i < 3 ; i++ ) scanf("%d",&nums[i]); //利用循环获取三个数字 for( i = 0 , max = nums [i]; i < 3 ; i++ ) //max 的初始值是 nums[0]，然后从第一个开始比较，若遇到更大的值就更新 max = MAX( max, nums[i] ); printf(" 这三个数字中最大的是: %d\n",max); return 0; }

3. 求 1 ~ 100 之间的所有偶数，每行输出 5 个。

答案 #include<stdio.h> int main() { int i, i2; //循环变量 i、控制换行的 i2 for( i = 1, i2 = 0 ; i <= 100 ; i++) //i 从 1 自增到 100 { if( i % 2 == 0 ) //只要 i 取余 2 为 0，是偶数，输出 { printf( "%d ", i); i2++; } if( i2 % 5 == 0 && i2 != 0) //输出 5 个，换行一次 printf("\n"); } return 0; }

4.编写程序，输入 30 个整数到一个一维数组中，求数组元素的最小值并输出。

答案 #include<stdio.h> #define MIN(x,y) ( (x) > (y) ) ? (y) : (x) int nums[30], i, min; int main() { for( i = 0 ; i < 30 ; i++) scanf("%d", &nums[i]); //获取 30 个整数 for( i = 0, min = nums[i] ; i < 30 ; i++ ) //从第一个开始比较，遇到更小的数字就更新 min 的值，遍历整个数组 min = MIN( min, nums[i] ); //进行一次比较 printf(" 这 30 个数中最小的是: %d\n", min); return 0; } //这道题和第二题的思路没什么区别
