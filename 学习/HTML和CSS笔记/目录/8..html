一、CSS 多列布局

多列布局模仿报纸的排版格式，它能将元素的内容分成很多列，并实现内容在列与列间自由流动，主要是四个 CSS 属性：
属性名属性值描述
columns长度值、数字指定每列最小宽度、列数
column-gap长度值指定列间距
column-rule与 border 相同定义列边框样式
column-spanall实现标题跨列效果
例子
columns:
写两个属性值，依次是每列最小宽度、列数，最小宽度的意思是，若窗口宽度小于 最小宽度 x 列数 时，会维持每列的最小宽度，减少列数。
column-gap:
指定列与列之间间距的大小。
column-rule:
定义列与列间分割线的样式，属性值与边框相同，但不一样的是，分割线不在文档流中，即不在元素中占据空间，相当于绝对定位。
