文档元素、嵌入元素

一、文档元素

文档元素是一堆基本没什么实际作用的元素，主要作用是便于管理页面结构，使编码语义化配合CSS使用,以下是主要的文档元素:

文档元素
序号元素名描述
1h1~h6分级表示标题，默认的字体加粗字号加大。
2header注意不是页面首部head，这个header只是body里的一个标签，意义是其内的内容是标题部分。
3footer页脚，用来写一些版权声明等等。
4nav表示导航元素，例如目录等。
5section重要概念或者小标题。
6article一段独立的内容，相当于一个内嵌的body,其内可含其它文档元素。
7address联系方式。
8aside注释栏，多放在页面的一侧，可有可无。
9hgroup组合标题，具体意义不明。
10details这是个有趣的元素，它能够创建一个可以展开的区域。
11summarydetails的配套子元素，是该元素内容的标题。
这些文档元素大多都只起划分结构的作用，其实也可以自己使用div和span来划分，但使用它们的好处是，这些文档元素是通用的，用它们写出来的页面更便 于其它编写人员看懂，也便于很久之后自己回头看懂页面结构。
details和summary示例: 
          <details>
            <summary>123</summary>
            7437290356y7810258163475612309-5748124356789423650123465178654p89235t7te hbdfrawSFCBFE
          </details>
在页面上的效果是这样的:
123 7437290356y7810258163475612309-5748124356789423650123465178654p89235t7te hbdfrawSFCBFE
很有趣的元素。

二、嵌入元素

嵌入元素
序号元素名作用
1img在页面中嵌入图片，属性:src、alt、width、height、ismap、usemap。
2map创建分区响应图，通过图片热点进行超链接，内嵌area元素，需要使用img的usemap属性进行关联。
3iframe在当前页面创建一个区域，显示另一个页面。
4embed引用插件，例如flash等。
5object是个通用引用元素，可以引用插件、图片、音频等，但是HTML4的标准。
6progress创建一个进度条，需要通过javescript控制内部的值，否则值不变化。
7meter反映范围内值的状态，外观也是一个进度条，但设置了最大最小值，有颜色提示。
嵌入元素的主要功能是在HTML中插入一些外部资源。
1.img 属性:src、alt、width、height、ismap、usemap。
src包含图片地址，alt是当图片无法显示时的注释。 ismap:可以显示图像每个位置的坐标，用法如下:
            <a href="本页面"><img src="图片地址" ismap></a>
效果如下:

每次点击过后，图片被点击处的坐标会显示在页面URL栏末尾处。
usemap:用来将图片与map元素关联。
2.map 作用是创建热点区域，即可以将图片划分为几个区域，每个区域都可以分别指定超链接。用法如下:
          <img src="图片地址" usemap="#m">
          <map name="m">
            <area shape="rect" coords="62,54,221,210" href="http://www.baidu.com" target="_blank">
            <area shape="circle" coords="300,420,500" href="http://Go-loneliness-of-man.github.io" target="_blank">
          </map>
可以看出，还有一个与其配合的area元素来确定区域，最后实现的效果是这样的:

可以发现，左边有一块方形区域链接到百度首页，右边有一块圆形区域链接到博客首页。
其中，area的shape属性确定形状，coords属性确定大小:
  1. rect,方形，coords="x1,y1,x2,y2",依次是图片左上角和右下角坐标。
  2. circle，圆形,coords="x1,y1,r",圆形坐标和半径。
  3. poligon，多边形，coords="x1,y1,x2,y2,x3...",多边形各顶点。
3.iframe iframe可以在一个html文档里内嵌一个html文档，即在页面中内嵌一个页面。用法如下:
            <a href="http://www.baidu.com" target="1">百度</a><br>
            <a href="http://Go-loneliness-of-man.github.io target="1">博客</a><br>
            <iframe src="https://www.imooc.com/" width="900px" height="800px" name="1"></iframe>
效果如下:
百度
博客

src指定此区域的初始链接页面，name属性为其它链接使用target属性指向此窗口创建条件。
4.embed 引用插件，例如一个flase视频，用法如下:
          <embed src="http://www.tudou.com/v/i4ZZvFwf;uI/&bid=-05&rpid=50797543&resourceId=50797543_05_05_99/v.swf" type="app
          lication/x-shockwave-flash" width="900px" height="700px"></embed>
效果如下:

type属性表示文件类型，src属性表示文件路径。由于H5已经可以结合JS编写越来越多的Web应用，所以将来引用的插件将越来越少。firefox不支持插件。
5.object 通用引用元素，HTML4的内容，在H5中已经被更多的语义化元素所分别代替。
6.progress 进度条，用法如下:
<progress value="40" max="100"></progress>
效果如下:

value是当前值，max是最大值，现在只是一个样式，要使其真正起作用，需要编写JS来控制其内部的值。IE9不支持。
7.meter 用进度条表示范围值，类似于progress，用法如下:
<meter value="50" min="30" max="100" low="40" high="80"></meter>
效果如下:

min和max确定值的范围，若value值不在范围内则空白，low和high划定正常范围，若不在范围内则会变色。IE浏览器不支持该元素。
