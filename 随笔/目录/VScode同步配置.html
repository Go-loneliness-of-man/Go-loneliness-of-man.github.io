VS code 同步设置

下载 setting sync 插件，shift + alt + u 上传当前插件及其配置，shift + alt + d 下载插件配置。

上传时会生成 Token 和 Gist，token 是个用于临时存储的东西，不重要，gist 是提取你配置时的密钥，要记好。当下载时，VS code 会自动弹出浏览器窗口要求你登录 github 账号创建 token，随便取个名字，在下面选取 gist，生成 token，将该 token 输入到 VS code 上方的框内，接着它会要求你输入 gist，该 gist 便是你上 传时生成的 gist。另外，每次使用完 token 都要记得重置插件设置，可以用 ctrl + shirt + p 调出命令面板输入 setting，下方会出现重置选项。
